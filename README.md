## עברית

אתר לסלון כלות **שש וארגמן**, הממוקם ביישוב חשמונאים, ישראל.  
האתר מציג את הסלון, גלריית כלות, מידע על הגעה ושאלות נפוצות.

### טכנולוגיות
- HTML5, CSS3, JavaScript (Vanilla)

### תכונות
- עיצוב רספונסיבי למובייל ודסקטופ
- גלריית תמונות בסגנון Pinterest עם Lightbox וזום
- תפריט ניווט חכם שנעלם בגלילה ומופיע בחזרה
- דף שאלות נפוצות (FAQ) עם אקורדיון
- תמיכה מלאה ב-RTL (עברית, ימין לשמאל)
- נגישות: ניווט מקלדת, תגיות ARIA, קישור דילוג לתוכן

## English

Website for **Shesh Ve'argaman** bridal salon, located in Hashmonaim, Israel.  
The site showcases the salon, a brides gallery, directions, and an FAQ page.

### Tech Stack
- HTML5, CSS3, JavaScript (Vanilla)

### Features
- Fully responsive design (mobile & desktop)
- Pinterest-style photo gallery with lightbox and zoom
- Smart sticky navbar (hides on scroll down, reappears on scroll up)
- FAQ page with accessible accordion
- Full RTL support for Hebrew
- Accessibility: keyboard navigation, ARIA labels, skip-to-content link
